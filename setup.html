Set up environment and getbarista

fork repo on github
add webhook: set http://getbarista.com:4000 as Payload URL
set application/x-www-form-urlencoded as content type
add public project from github at getbarista
set django settings module on project dashboard
Add https://kavahq.com/api/barista/post_deploy/ to Post-deployment URLs on the getbarista project dashboard
clone project locally
install virtualenv
install requirements
make initial commit

Set up contacts

set your user.email in git. man git-config for details
set up your jabber
enter your email on the trac`s Preferences
add your ssh public key into https://kavahq.com/account/
